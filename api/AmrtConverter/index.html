AmrtConverter 类¶

概述¶

AmrtConverter 类是 AMRT SDK 的核心转换器，负责处理各种 3D 模型格式之间的转换。

命名空间: amrt
头文件: <amrt/converter.h>

类定义¶

class AmrtConverter {
public:
    // 构造函数
    AmrtConverter();
    ~AmrtConverter();

    // 转换方法
    bool convert(const std::string& inputPath, 
                 const std::string& outputPath);

    bool convertWithProgress(const std::string& inputPath, 
                            const std::string& outputPath,
                            ProgressCallback callback);

    // 配置方法
    void setOptions(const ConvertOptions& options);
    ConvertOptions getOptions() const;

    // 状态查询
    std::string getLastError() const;
    bool isConverting() const;
};

构造函数¶

AmrtConverter()¶

创建一个新的转换器实例。

示例: 

amrt::AmrtConverter converter;

公共方法¶

convert()¶

执行基本的格式转换。

函数签名: 

bool convert(const std::string& inputPath, 
             const std::string& outputPath)

参数: - inputPath - 输入文件的完整路径 - outputPath - 输出文件的完整路径

返回值: - true - 转换成功 - false - 转换失败（调用 getLastError() 获取错误信息）

支持的格式: - 输入: .gltf, .glb, .fbx, .obj, .dae - 输出: .amrt

示例: 

#include <amrt/converter.h>

int main() {
    amrt::AmrtConverter converter;

    if (converter.convert("model.gltf", "output.amrt")) {
        std::cout << "转换成功!" << std::endl;
    } else {
        std::cerr << "错误: " << converter.getLastError() << std::endl;
    }

    return 0;
}

convertWithProgress()¶

执行带进度回调的格式转换。

函数签名: 

bool convertWithProgress(const std::string& inputPath, 
                        const std::string& outputPath,
                        ProgressCallback callback)

参数: - inputPath - 输入文件路径 - outputPath - 输出文件路径
- callback - 进度回调函数，签名为 void(float progress, const std::string& stage)

返回值: - true - 转换成功 - false - 转换失败

示例: 

converter.convertWithProgress(
    "model.gltf", 
    "output.amrt",
    [](float progress, const std::string& stage) {
        std::cout << stage << ": " << (progress * 100) << "%" << std::endl;
    }
);

setOptions()¶

设置转换选项。

函数签名: 

void setOptions(const ConvertOptions& options)

参数: - options - 转换选项结构体

示例: 

amrt::ConvertOptions opts;
opts.compressTextures = true;
opts.optimizeMeshes = true;
opts.mergeVertices = true;
opts.generateNormals = false;

converter.setOptions(opts);

getOptions()¶

获取当前的转换选项。

函数签名: 

ConvertOptions getOptions() const

返回值: - 当前使用的转换选项

getLastError()¶

获取最后一次操作的错误信息。

函数签名: 

std::string getLastError() const

返回值: - 错误描述字符串，如果没有错误则返回空字符串

示例: 

if (!converter.convert(input, output)) {
    std::cerr << "转换失败: " << converter.getLastError() << std::endl;
}

isConverting()¶

检查转换器是否正在执行转换操作。

函数签名: 

bool isConverting() const

返回值: - true - 正在转换 - false - 空闲状态

完整示例¶

#include <amrt/converter.h>
#include <iostream>

int main() {
    // 创建转换器
    amrt::AmrtConverter converter;

    // 配置选项
    amrt::ConvertOptions options;
    options.compressTextures = true;
    options.optimizeMeshes = true;
    converter.setOptions(options);

    // 执行转换（带进度）
    bool success = converter.convertWithProgress(
        "assets/scene.gltf",
        "output/scene.amrt",
        [](float progress, const std::string& stage) {
            printf("\r[%s] %.1f%%", stage.c_str(), progress * 100);
            fflush(stdout);
        }
    );

    if (success) {
        std::cout << "\n✅ 转换完成!" << std::endl;
    } else {
        std::cerr << "\n❌ 转换失败: " 
                  << converter.getLastError() << std::endl;
        return 1;
    }

    return 0;
}
