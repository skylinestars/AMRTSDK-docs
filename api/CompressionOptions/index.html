类型: bool

compressNormals¶

类型: bool

压缩法线数据

compressTextures¶

类型: bool

压缩纹理

compressUVs¶

类型: bool

压缩UV数据

compressVertices¶

类型: bool

压缩顶点数据

dracoCompressionLevel¶

类型: int

Draco压缩级别 [0-10]，越高压缩率越高但速度越慢

dracoQuantizationBits¶

类型: int

Draco量化位数 [8-30]，越少压缩率越高但精度越低

enableDracoCompression¶

类型: bool

启用Draco几何压缩

enableLZ4Compression¶

类型: bool

启用LZ4数据压缩
